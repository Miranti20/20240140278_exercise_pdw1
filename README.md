Tugas Pengembangan Desain Web - Exercise 1

Website ini dibuat untuk memenuhi tugas mata kuliah Pengembangan Desain Web. Tema yang diangkat pada project website statis ini adalah "Profil Kedai Kopi (Kedai Kopi Senja)". Website ini menampilkan informasi singkat mengenai kedai kopi, menu andalan, dan dilengkapi dengan interaksi dasar menggunakan JavaScript.
